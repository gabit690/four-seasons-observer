# Four seasons observer

Zona de detección scrolleable que produce cambios relacionados a las estaciones del año.

## Objetivo

Poner en práctica la API [Intersection Observer](https://developer.mozilla.org/en-US/docs/Web/API/Intersection_Observer_API) de JavaScript.

## Tecnologías

- HTML5
- CSS3
- JavaScript
